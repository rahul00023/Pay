Aufkleber für Ihr Geschäft bestellen

Lassen Sie Ihre Kunden wissen, dass Sie Google Pay akzeptieren, indem Sie Aufkleber an der Tür, an der Kasse und am Zahlungsterminal anbringen. Füllen Sie einfach dieses Formular aus und wir senden Ihnen innerhalb von 8 bis 12 Werktagen ein Aufkleberset.

* Pflichtfelder

Fertig

Wenn Sie eine gültige Postanschrift angegeben haben, erhalten Sie die Aufkleber innerhalb von 7 bis 10 Werktagen.

Ein Fehler ist aufgetreten.

Es gab ein Kommunikationsproblem zwischen den Servern. Bitte versuchen Sie es noch einmal.

Bitte geben Sie eine gültige E-Mail-Adresse ein
Bitte geben Sie eine gültige Postleitzahl ein.
Bitte geben Sie eine Bestellung auf (bis zu 100 Aufklebersets).

Ein Set umfasst 2 doppelseitige Aufkleber für Fenster, 4 Aufkleber für die Kasse und 8 Aufkleber für das Zahlungsterminal.

Bitte akzeptieren Sie die Nutzungsbedingungen, bevor Sie fortfahren.
