ブランドの親近感を高める

Google Pay でよりスマートなショッピング体験を提供しましょう。

API だけあればOK

Google Pay API for Passes を使用すれば、ポイント プログラム、ギフトカード、特典を管理して宣伝することで、何百万人もの Google ユーザーとつながることができます。位置情報ベースの通知、リアルタイム更新、メッセージで、いつでもどこでも顧客とのエンゲージメントが可能です。

API ドキュメント﻿

登録

顧客は異なる端末間で自動的にログインできるので、パスワード忘れのリスクやカート放棄率が下がります。

詳細﻿

特典をリンク

Google Payにポイントカードを保存した顧客に、パーソナライズした特典をお届けすることができます。

特典を作成する﻿

スマートタップ

ユーザーは NFC 対応の決済端末にスマートフォンをかざすだけで、Google Pay に保存されているポイントカード、ギフトカードやクーポンを利用できます。
